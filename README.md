# Avocodos
Avocodos is a completely web3-based social media and learning platform.

## Features
- Create and view courses
- Get rewareded with NFTs for completing courses
- Create posts
- Create comments
- Create reactions
- Create communities
- Follow other users
- Community moderation features
- Community flair/badges

## Technologies
- Next.js
- TailwindCSS
- Prisma
- Aptos
- Aptos-ts-sdk
